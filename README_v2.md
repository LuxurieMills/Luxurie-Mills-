# Single-Cell RNA-seq Analysis Pipeline Using Scanpy

## Overview

This project demonstrates a single-cell RNA sequencing analysis workflow using Python and Scanpy.

## Objective

Analyze gene expression data at the single-cell level to identify cell populations and visualize biological patterns.

## Tools

- Python
- Scanpy
- Pandas
- NumPy
- Jupyter Notebook
- GitHub

## Workflow

1. Load single-cell RNA-seq data
2. Perform quality control
3. Normalize gene expression data
4. Perform dimensionality reduction
5. Cluster cells
6. Visualize results

Project currently under development.
